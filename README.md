# 💬 ChatHaven

**ChatHaven** is a full-stack real-time chat application that supports text, image sharing, and advanced WebRTC-based audio/video calling — all wrapped in a modern, themeable UI.

---

## 🚀 Features

- 🔐 User Authentication (JWT + Cookies)
- 💬 One-on-One Chat (Text + Image)
- 🎥 Audio & Video Calling (WebRTC)
- 📡 Real-time Messaging (Socket.IO)
- 🌐 Online/Offline User Presence
- 🎨 Light/Dark & Themed UI (TailwindCSS + DaisyUI)
- 📁 Profile Picture Upload (Cloudinary)
- 🧠 Zustand for Global State
- 📱 Responsive + Mobile-Ready

---

## 🛠️ Tech Stack

### 🔹 Frontend
- React + Vite
- Zustand (state management)
- TailwindCSS + DaisyUI
- WebRTC (native API)
- Socket.IO (client)
- React Router DOM

### 🔹 Backend
- Node.js + Express
- MongoDB + Mongoose
- Socket.IO (server)
- Cloudinary (image upload)
- JWT for Auth
